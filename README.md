## Getting Started

Install the following to run SVG Logo Maker

### Prerequisites

Install <i>npm i inquirer</i>.
  ```sh
  npm i inquirer@8.2.4
  ```

Install <i>Jest.</i>
  ```sh
  npm install --save-dev jest
  ```

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run
    ```
    node index.js
    ```
